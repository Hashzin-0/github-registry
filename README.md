# GitHub Registry MCP Server

MCP Server para gerenciar Skills, Agentes e MCPs em repositórios GitHub.

## Funcionalidades

### Operações GitHub Básicas
- `read_file`
- `write_file`
- `list_directory`

### Operações de Registry
- `init`
- `save`
- `search`
- `list`
- `get_index`
- `dynamic_executor`

## Instalação Rápida

```bash
npx ghr-mcp
```

Esse comando configura automaticamente o MCP remoto no OpenCode.
